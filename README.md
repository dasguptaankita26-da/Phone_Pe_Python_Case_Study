# 📱 PhonePe Transaction Analysis using Python

## 📌 Project Overview

This project analyzes **PhonePe digital payment transaction data** using Python to uncover trends in transaction volume, transaction value, state-wise performance, and user adoption across India. The analysis provides insights into digital payment growth through data cleaning, visualization, and exploratory data analysis (EDA).

---

## 🎯 Objectives

* Analyze transaction volume and transaction value.
* Compare digital payment adoption across Indian states.
* Identify top-performing states and transaction categories.
* Visualize payment trends using interactive charts.
* Generate business insights from transaction data.

---
